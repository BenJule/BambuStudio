BenJule BambuStudio APT Repository

This repository provides .deb packages for BambuStudio — a fork of bambulab/BambuStudio.

Installation

One-liner (recommended)

curl -fsSL https://benjule.github.io/BambuStudio/KEY.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/bambustudio.gpg
echo "deb [signed-by=/etc/apt/trusted.gpg.d/bambustudio.gpg] https://benjule.github.io/BambuStudio stable main" | sudo tee /etc/apt/sources.list.d/bambustudio.list
sudo apt update
sudo apt install bambu-studio

Upgrade

sudo apt update && sudo apt upgrade bambu-studio

Remove

sudo apt remove bambu-studio
sudo rm /etc/apt/sources.list.d/bambustudio.list /etc/apt/trusted.gpg.d/bambustudio.gpg
